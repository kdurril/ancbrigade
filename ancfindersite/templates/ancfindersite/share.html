We Need Your Help!

This project is only as good as the data in it, so we need your help to make sure that we have the most up-to-date information. There are several ways that you can chip in:

Submit Commissioner Information

If you spot anything incorrect on this site, you can update it! Follow the edit links where you see commissioner information on the site.

Submit Some Documents

Using our document uploader, you can share documents in 3 different ways: uploading PDFs, sharing a link to a PDF on another website, or just pasting the text of a document.

We're looking for any kind of ANC document: agendas, minutes, resolutions, grants, etc. You can also edit previously-uploaded documents to add or correct information aobut them.

Correct Us!

While we're trying our best to pull data only from sources that we trust, it's always possible that something goes wrong along the way. If you see information that needs fixing, and you can't fix it through either of the above methods, let us know! We want to make sure that we do this right.

Open a Pull Request on Github

This site is an open-source project. Have a suggestion for the code behind it? Let us know! The link to our repo is down there at the bottom of the page.
